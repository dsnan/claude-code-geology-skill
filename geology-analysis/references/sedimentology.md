# 地层学与沉积学参考

## 沉积相分析流程

观察 → 描述 → 分类命名 → 水动力解释 → 区域对比 → 沉积模式构建

## Walther 相律

> "只有横向相邻的沉积环境才能在垂向序列中叠置出现"

垂向序列 = 横向环境迁移的时间记录。用这个原理从钻孔/剖面的垂向序列反推横向古地理格局。

## 层序地层学关键界面

| 界面 | 标志 | 意义 |
|------|------|------|
| 层序边界 (SB) | 不整合面、暴露面 | 基准面下降 |
| 首次洪泛面 (FFS) | 泥岩覆盖砂岩 | 基准面开始上升 |
| 最大洪泛面 (MFS) | 凝缩段、暗色泥岩 | 基准面最高点 |

## 常见沉积构造及水动力解释

| 构造 | 水流体制 | 环境 |
|------|----------|------|
| 水平层理 | 静水 | 湖/深海 |
| 波状层理 | 波浪 | 滨岸 |
| 槽状交错层理 | 单向流 | 河流 |
| 板状交错层理 | 单向流/潮汐 | 河流/潮汐 |
| 丘状交错层理 | 风暴浪 | 浅海 |
| 粒序层理 | 浊流 | 深海/湖底 |
| 泥裂 | 暴露 | 潮上/湖岸 |
| 雨痕 | 暴露 | 干旱区 |
| 生物扰动 | 富氧底质 | 多种 |

## 碳酸盐岩分类 (Dunham, 1962)

| 类型 | 特征 | 沉积环境 |
|------|------|----------|
| Mudstone | 泥质支撑，<10%颗粒 | 低能、潟湖 |
| Wackestone | 泥质支撑，>10%颗粒 | 中低能 |
| Packstone | 颗粒支撑，含泥 | 中能 |
| Grainstone | 颗粒支撑，无泥 | 高能、浅滩 |
| Boundstone | 生物粘结 | 礁体 |
| Floatstone | 大颗粒浮于泥中 | 滑塌/碎屑流 |
| Rudstone | 大颗粒相互接触 | 风暴/高能 |

## 碎屑岩成分成熟度

- **不成熟**：大量长石+岩屑 → 近源、快速剥蚀、干旱/寒冷气候
- **成熟**：石英为主 → 远源、长期风化/搬运、湿热气候
- **超成熟**：纯石英 → 克拉通、多旋回再循环

## 常见沉积环境识别标志

| 环境 | 岩性标志 | 化石标志 | 构造标志 |
|------|----------|----------|----------|
| 河流相 | 砂岩、砾岩、向上变细序列 | 淡水生物、植物碎片 | 槽状交错层理、底冲刷 |
| 湖泊相 | 泥岩、粉砂岩、泥灰岩 | 淡水介形虫、盘星藻、孢粉 | 水平层理、纹层 |
| 三角洲相 | 砂泥互层、向上变粗序列 | 海陆混生化石 | 前积层理、生物扰动 |
| 浅海相 | 碳酸盐岩、砂岩 | 海相化石（有孔虫等） | 丘状交错层理、风暴层 |
| 深海相 | 远洋软泥、燧石 | 放射虫、硅藻 | 水平层理、浊积序列 |
| 风成沉积 | 中-细砂岩、分选极好 | 罕见 | 大型高角度交错层理 |
| 冰川沉积 | 混杂堆积、冰碛岩 | 耐寒孢粉 | 无层理、擦痕砾石 |
| 蒸发环境 | 蒸发岩（石膏、盐岩） | 罕见 | 盐构造、泥裂 |
