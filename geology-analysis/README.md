# claude-code-geology-skill
 Claude Code 自定义技能：综合地质分析（沉积学、地层学、构造地质学、古环境重建、地貌学、矿产资源评估）。支持孢粉/粒度/地球化学/测年等代用指标解读、沉积相分析、成矿系统评估、地质报告撰写。
 ## 安装

  将 `geology-analysis.md` 复制到 `~/.claude/skills/` 目录，重启 Claude Code 即可。

  ```bash
  mkdir -p ~/.claude/skills
  cp geology-analysis.md ~/.claude/skills/

使用方法：
在 Claude Code 中输入 /geology-analysis，或直接描述你的地质问题。
