# 地貌学参考

## 地貌系统核心方程

> 地貌 = f（构造抬升, 气候侵蚀, 时间）

分析地貌时永远从这个三维框架出发，不能只看单一因素。

## 地貌旋回 (Davis, 1899)

| 阶段 | 特征 | 地形 |
|------|------|------|
| 幼年期 | 抬升主导、河流下切为主 | V形谷、峡谷、瀑布 |
| 壮年期 | 下切与侧蚀平衡 | 宽谷、曲流、山脊尖锐 |
| 老年期 | 侧蚀和剥蚀为主 | 准平原、残丘 |

注意：Davis 模型是理想化的——现实中构造活动往往是多期次的，地形可能是多旋回叠加的结果。

## 河流地貌定量指标

这些指标用于从地形数据中提取构造活动信号：

| 参数 | 含义 | 构造/气候指示 |
|------|------|-------------|
| 河道坡度指数 (SL) | 河段坡度异常 | SL 异常→构造活动/岩性变化 |
| 流域不对称因子 (AF) | 左右岸面积比 | AF≠50→构造掀斜 |
| 山前曲折度 (SMF) | 山前是否平直 | SMF<1.4→活动构造 |
| 谷底宽高比 (Vf) | V形/U形谷 | Vf<1→下切 > 拓宽（抬升区） |
| 高程积分 (HI) | 残余地形体积 | HI>0.5→幼年/壮年，HI<0.3→老年 |

## 风成地貌

| 类型 | 形态 | 形成条件 |
|------|------|----------|
| 新月形沙丘 | 新月形、角指向下风 | 单向风、沙源有限 |
| 线形沙丘 | 平行风向的长脊 | 双向风、沙源充足 |
| 星状沙丘 | 多臂、中心高耸 | 多风向 |
| 抛物线沙丘 | U形、角指向上风 | 植被部分固定 |
| 黄土 | 均质粉砂沉积 | 风力搬运、半干旱草原 |

## 冰川地貌

| 类型 | 形态 | 过程 |
|------|------|------|
| U形谷 | 槽状、陡壁平底 | 冰川侵蚀 |
| 冰斗 (Cirque) | 椅状洼地 | 源头侵蚀 |
| 角峰 (Horn) | 金字塔形山峰 | 多方向冰斗汇合 |
| 终碛 (Terminal Moraine) | 弧形脊 | 冰川前端堆积 |
| 羊背石 (Roche Moutonnée) | 不对称岩丘 | 磨蚀+拔蚀 |

## 活动构造地貌识别标志

| 标志 | 说明 |
|------|------|
| 断层陡坎 | 保存完好的线性陡坎→年轻活动 |
| 水系位错 | 河道系统偏移→走滑活动 |
| 河流阶地变形 | 同级阶地高差异常→差异抬升 |
| 冲积扇叠置 | 扇顶向山迁移→山前断层活动 |
| 夷平面解体 | 同级夷平面高程差→后期构造 |
