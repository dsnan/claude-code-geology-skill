# 地球化学参考

## TAS 图分类（火成岩全碱-硅图解）

用于火山岩和深成岩的化学分类，是最基础的火成岩命名工具：

| 岩石类型 | SiO₂ 范围 | Na₂O+K₂O 特征 |
|----------|----------|---------------|
| 超基性岩 | <45% | <2% |
| 基性岩（玄武岩/辉长岩） | 45-52% | 2-5% |
| 中性岩（安山岩/闪长岩） | 52-63% | 5-8% |
| 酸性岩（流纹岩/花岗岩） | >63% | >7% |

细分需用具体 TAS 网格坐标。TAS 图不适用于碱性岩和超基性岩的细分。

## 稀土元素 (REE) 配分模式

用球粒陨石标准化后作图，根据曲线的倾斜度和异常解释成因：

| 特征 | 含义 |
|------|------|
| LREE 富集 (La/Yb)N > 1 | 源区含石榴石残留 / 低程度部分熔融 / 地壳混染 |
| HREE 亏损 (Gd/Yb)N > 1 | 源区有石榴石残留（高压条件） |
| Eu 负异常 (Eu/Eu* < 0.8) | 斜长石分离结晶或源区斜长石残留 |
| Eu 正异常 (Eu/Eu* > 1.2) | 斜长石堆晶 |
| 平坦型 (La/Yb)N ≈ 1 | 亏损地幔来源（MORB） |

解释原则：REE 配分模式 + 微量元素蛛网图 + 同位素数据三者结合，单一稀土图不能定论。

## 化学风化指数 (CIA)

CIA = [Al₂O₃ / (Al₂O₃ + CaO* + Na₂O + K₂O)] × 100 （摩尔比，CaO* 仅硅酸盐中的钙）

| CIA 值 | 风化程度 | 气候指示 |
|--------|----------|----------|
| <50 | 未风化 | 寒冷/干旱 |
| 50-65 | 初级风化 | 温带、中等降水 |
| 65-85 | 中等风化 | 暖温带、较高降水 |
| >85 | 强烈风化 | 热带、高降水 |
| ~100 | 极端风化（铝土矿） | 强热带风化 |

用于沉积物源区气候重建——从碎屑沉积岩/泥岩的化学组成反推源区当时的风化环境。

## 常用同位素体系

| 体系 | 示踪对象 | 关键端元 |
|------|----------|----------|
| ⁸⁷Sr/⁸⁶Sr | 地壳 vs 地幔来源 | 亏损地幔 ~0.7025，大陆地壳 >0.710 |
| ¹⁴³Nd/¹⁴⁴Nd (εNd) | 地壳 vs 地幔来源 | εNd>0 亏损地幔，εNd<0 富集源/地壳 |
| ²⁰⁶Pb/²⁰⁴Pb | 地壳演化史 | 上地壳高μ值（高U/Pb），下地壳低μ |
| δ¹³C | 碳循环、有机 vs 无机碳 | 海相碳酸盐 ~0‰，有机碳 -25‰，大气 CO₂ -7‰ |
| δ¹⁸O | 温度、降水来源、成岩蚀变 | 海水 ~0‰，大气降水负值（越冷越负），岩浆 ~5.5-7‰ |
| δ³⁴S | 氧化还原、生物硫酸盐还原 | 海相硫酸盐 +20‰，硫化物 可正可负 |

### Sr-Nd 同位素联合图解

最常用的源区判别工具——同时用 ⁸⁷Sr/⁸⁶Sr 和 εNd(t)：
- 亏损象限（高 εNd, 低 Sr）：亏损地幔
- 富集象限（低 εNd, 高 Sr）：上地壳 / 富集地幔
- 地幔排列 (Mantle Array)：亏损地幔到富集地幔的趋势线
- 偏离地幔排列的样品 → 地壳混染信号

## 蛛网图异常解读

微量元素原始地幔标准化蛛网图中：

| 异常 | 矿物控制 |
|------|----------|
| Nb-Ta 负异常 | 金红石/榍石 / 地壳信号 / 俯冲带特征 |
| Ti 负异常 | Fe-Ti 氧化物分离 / 岛弧岩浆 |
| Sr 正异常 | 斜长石堆晶 |
| P 负异常 | 磷灰石分离 |
| Zr-Hf 正异常 | 锆石堆晶 |
| Pb 正异常 | 地壳混染（上地壳高Pb特征） |
