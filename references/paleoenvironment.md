# 古环境重建参考

## 代用指标解读框架

### 生物指标

- **孢粉 (Pollen)**：植被类型、温度、降水
  - A/C 比值（Artemisia/Chenopodiaceae）：干旱区有效湿度指标
  - AP/NAP 比值：森林覆盖度
  - 热带/温带成分比：温度指示
- **硅藻 (Diatom)**：水体盐度、pH、营养状态
- **介形虫 (Ostracod)**：水体盐度、深度、温度
- **炭屑 (Charcoal)**：火灾频率和强度

### 物理指标

- **粒度 (Grain Size)**：水动力条件、风成输入
  - >63μm 砂级：强水动力/风力搬运
  - 4-63μm 粉砂：河流/风力悬移
  - <4μm 黏土：静水沉积
  - 中值粒径趋势：水动力变化方向
- **磁化率 (Magnetic Susceptibility)**：碎屑输入量、氧化还原状态
- **色度 (Color)**：有机质含量、氧化还原状态

### 化学指标

- **总有机碳 (TOC)**：初级生产力、保存条件
- **C/N 比**：有机质来源（陆源 C/N>15 vs 水生 C/N<10）
- **碳酸盐含量**：化学沉淀/生物壳体贡献
- **δ¹³C**：C3/C4 植被比例、碳循环
- **δ¹⁸O**：温度、降水来源（大气降水 vs 蒸发）
- **微量元素**：风化强度（CIA）、氧化还原（V/Cr, Ni/Co）

## 气候带-植被对应（中国北方）

| 气候带 | 年降水 | 典型孢粉组合 |
|--------|--------|-------------|
| 荒漠 | <100mm | Ephedra+Nitraria+Amaranthaceae 优势 |
| 荒漠草原 | 100-200mm | Artemisia+Amaranthaceae+Ephedra |
| 典型草原 | 200-400mm | Artemisia+Poaceae 优势 |
| 森林草原 | 400-600mm | Pinus+Betula+Artemisia+Poaceae |
| 森林 | >600mm | Pinus+Betula+Quercus 优势 |

## 孢粉-古气候指标速查

| 指标 | 说明 | 应用方向 |
|------|------|----------|
| A/C 比值 | 蒿属/藜科比 | 干旱区有效湿度 |
| AP/NAP 比值 | 木本/草本花粉比 | 植被覆盖类型 |
| 针叶/阔叶比 | 温度指示 | 海拔/纬度温度梯度 |
| Pediastrum 丰度 | 淡水藻类 | 湖泊水位/面积 |
| Ephedra 波动 | 荒漠灌丛 | 干旱化程度 |
| 炭屑 (Charcoal) | 火灾指标 | 自然/人类火灾 |

## 湖泊演化阶段模型

| 阶段 | 沉积特征 | 生物标志 | 环境意义 |
|------|----------|----------|----------|
| 深水湖 | 纹层泥岩 | 鱼化石、远洋硅藻 | 湿润期、高水位 |
| 湖泛期 | 暗色泥岩 | 介形虫、腹足类 | 水位上升 |
| 浅水湖 | 粉砂质泥岩 | 底栖生物、盘星藻 | 半干旱、中水位 |
| 沼泽 | 泥炭/碳质泥岩 | 莎草科、香蒲 | 浅水、富营养 |
| 干涸 | 蒸发岩/钙积层 | 罕见 | 极端干旱 |

## 常见测年方法

| 方法 | 材料 | 范围 | 精度 |
|------|------|------|------|
| ¹⁴C (AMS) | 有机质、碳酸盐 | 0-50 ka | ±20-200 yr |
| OSL | 石英/长石颗粒 | 0-300 ka | ±5-10% |
| U-Th | 碳酸盐 | 1-500 ka | ±1% |
| 磁性地层 | 沉积序列 | 整个第四纪 | 取决于极性转换 |
| 孢粉地层 | 沉积序列 | 新生代 | 取决于区域对比 |
| 宇宙成因核素 | 地表岩石 | 1-10⁶ yr | ±5-15% |
| 锆石 U-Pb | 锆石 | 全地质历史 | ±1-2% |
| Ar-Ar | 火山岩矿物 | 新生代-中生代 | ±0.1-1% |
