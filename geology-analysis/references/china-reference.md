# 中国区域地质参考

## 中国主要构造域

| 构造域 | 位置 | 主要特征 | 关键事件 |
|--------|------|----------|----------|
| 华北克拉通 | 中国北部 | 古老基底，稳定地块 | 中生代破坏/减薄 |
| 扬子克拉通 | 华南 | 新元古代基底 | 中生代褶皱/逆冲 |
| 塔里木克拉通 | 新疆南部 | 前寒武基底 | 新生代盆地沉降 |
| 青藏高原 | 西南 | 新生代造山带 | 印度-欧亚碰撞 |
| 天山-兴蒙造山带 | 北部 | 古生代造山带 | 古亚洲洋闭合 |
| 秦岭-大别造山带 | 中部 | 三叠纪碰撞带 | 华北-扬子碰撞 |
| 滨太平洋构造域 | 东部 | 中生代岩浆-伸展 | 太平洋俯冲 |

## 中国地貌三级阶梯

| 阶梯 | 海拔 | 主要地貌单元 |
|------|------|------------|
| 第一阶梯 | >4000m | 青藏高原 |
| 第二阶梯 | 1000-4000m | 内蒙古高原、黄土高原、云贵高原、塔里木、准噶尔、四川盆地 |
| 第三阶梯 | <1000m | 东北平原、华北平原、长江中下游平原、东南丘陵 |

分析中涉及区域背景时，先确定所处阶梯和构造域，再讨论具体问题。

## 中国常见构造问题

| 问题 | 区域 | 关键分析点 |
|------|------|-----------|
| 青藏高原隆升 | 西南 | 碰撞时序、地壳缩短量、地表抬升史 |
| 华北克拉通破坏 | 华北 | 岩石圈减薄、中生代岩浆作用 |
| 华南中生代变形 | 华南 | 多期褶皱叠加、花岗岩时代 |
| 天山再生造山 | 新疆 | 陆内变形、盆山耦合 |
| 郯庐断裂 | 东部 | 大规模平移、多期活动 |
